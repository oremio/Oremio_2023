Java语言和C++语言中都可以声明常量。

Java使用关键字 final，C++使用关键字 const。在C++11之后，还有一个关键字 constexpr 用于声明编译器常量

java的常量与C++的常量有什么区别呢？

java:

final int a;  // 声明常量 a。final 值声明的时候可以不做初始化

a = 10;     // 第一次赋值时初始化

a = 12;     // 错误。不允许二次赋值

c++:

const int a = 10; // 声明常量a

const int b;      // 错误。命名常量必须在声明的同时初始化

a = 42;        // 错误。常量一旦声明，就不能再修改
