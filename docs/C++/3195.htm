什么是 UML？

UML（Unified Modeling Language）是一种用于面向对象软件开发的标准建模语言。
提供了一种图形化的表示方式，可以用于描述软件系统的结构、行为和交互等方面的内容，从而使软件开发者能够更好地理解和沟通软件系统的设计和实现。

包括类图、用例图、序列图、活动图、状态图。
