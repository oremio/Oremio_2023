什么是抖动？

抖动（thrashing）是指系统因频繁地进行页面置换而导致性能下降的现象。

抖动一般是由于进程太多，内存不足导致的。为了避免抖动，可以采取一些措施，如增加内存容量、优化进程调度算法、优化页面置换算法等。
