什么是虚函数？

在 C++ 中指的是可以被子类中同名的方法继承、覆盖的类的成员方法；是面向对象程序设计中运行时多态的重要组成部分；虚函数给出了一个函数的定义，但是该目标的具体指向在编译期间可能无法确定。

虚函数在 C++ 中通过虚函数表来实现：虚函数表包含了一个类所有的虚函数的地址，每个类的所有对象共享一张虚函数表；而在有虚函数的类对象中，它内存空间的头部会有一个虚函数表指针；这样即使通过父类指针引用子类对象，也可以通过虚函数表找到实际应当调用的函数，实现了运行时多态。

但是并不是所有的虚函数调用都是运行时确定的 —— 只有引用和指针的虚函数调用是运行时确定；

虚函数在设计模式中也有广泛的应用。
