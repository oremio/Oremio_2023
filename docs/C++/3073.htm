启动计算机的过程？

 - 加电自检（POST）：通电时主板进行，检测硬件是否正常工作

 - BIOS 自检：主板将权限交给 BIOS，BIOS 检查计算机基本配置信息和时钟

 - 加载引导程序：控制权移交给了硬盘上的引导程序（Boot Loader），该程序会加载操作系统的内核文件到内存中，并转交控制权

  ○ BIOS 加载主引导扇区（MBR），MBR 包含了分区表和引导程序

  ○ 执行引导程序：查找内核位置，并加载到指定地址

  ○ 移交控制权给内核

 - 操作系统启动：内核加载后会进行初始化，启动各个子系统，为用户程序的运行做准备

 - 登录用户：用户程序运行
