DEFINE 和 const（constexpr）的区别？

DEFINE 是 C/C++ 语言中的宏，可以用于声明字面量常量和函数；而 const 是 C/C++ 中声明常量的关键字；而 constexpr 是 C++11 后用于声明编译期可以求值的常量。

在声明常量的时候：

 - Define 不做类型检查，只是进行单独的字符串替换；但是支持一些高级的用法，比如字符串拼接等。也可以定义函数式的宏，在各种测试框架和反射框架中用的比较多

 - Const 声明了常量，储存在常量数据区域，但是只能声明常量

 - Constexpr 声明的常量会在编译时求值，并根据编译器的选择来决定将所有的出现位置替换为结果的字面量，或者像 const 一样存储到只读数据区。并且 constexpr 可以用于 C++ 模板编程。

new/delete 和 malloc/free 的区别？

 - 指定分配内存的大小：new 不需要，只需要类型；但 malloc 需要手动声明需要分配的内存大小

 - 类型安全：new 返回声明类型的指针，类型安全；malloc 只返回 void* 指针，需要使用强制类型转换

 - 对于对象的行为不同：new 会调用构造函数和析构函数；malloc 不会。

 - 失败时的行为不同：new 会抛出 bad_alloc，malloc 则返回空指针
