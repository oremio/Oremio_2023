系统调用的过程？

 - 用户程序通过访管指令发出系统调用请求，控制权转移到内核态

 - OS 保存进程上下文

 - OS 根据系统调用号和参数来确定用户请求的服务/资源，并在内核态下执行相应操作

 - OS 将执行结果保存在内存/寄存器中以返回用户程序

 - 控制权还给用户进程，继续执行
