哈希复杂度？ASL？

在平均情况下，哈希表的复杂度是 O(1)；最差会达到 O(n)，通常发生在哈希函数设计不合理或者哈希表装填因子过高的场合。

ASL是指平均查找长度（Average Search Length），是在查找某个元素时需要访问的节点数的期望值。它是衡量查找效率的重要指标之一，可以用来评估哈希表的性能。对于一个包含n个元素的哈希表，ASL的计算公式为：

ASL = (成功查找所需的节点数 + 失败查找所需的节点数) / n
