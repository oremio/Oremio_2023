堆和栈之间的区别？

 - 堆内存需要手动管理分配和释放，而栈中的内存由 OS 自行管理。

 - 堆大，而栈往往比较小

 - 频繁分配/释放堆内存会产生大量的碎片，而栈则不存在这个问题

 - 堆向着高地址方向生长，而栈向着低地址方向生长

 - 栈可以静态分配（如局部变量）和动态分配（alloca），都能在编译时由编译器确实释放；而堆只能动态分配且必须手动释放

 - 栈的分配效率更高，是硬件支持的；而堆的分配则更复杂，需要找到足够的空间并由程序分配，效率更低
