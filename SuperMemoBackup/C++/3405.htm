通过基类指针访问同名函数的示例
 

class P           { public: f(){…} }; //父类

class C: public P { public: f(){…} }; //子类

main () {

    P* ptr;   P p;   C c;

    ptr = &p;

    ptr->f();      // 调用P::f()

 

    ptr=&c;

    ptr->f();      // 调用P::f()

}

 

 

说明: 指针是什么类型，就调什么类型

 

 

动态联编的简单示例-指针形式
 

class P           { public: virtual f(){…} }; //父类

class C: public P { public: f(){…} }; //子类，f自动virtual

main () {

    P* ptr;   P p;   C c;

    ptr = &p;

    ptr->f();    //调用P::f()

 

    ptr=&c;

    ptr->f();    //调用C::f()

}

 

说明: 函数虚，不看指针看真对象
